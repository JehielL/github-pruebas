CURSO JAVA

PRACTICA GIT COMMITS

Prueba de clonar repositorios, hacer push y realizar cambios.

  • Pasos Para clonar

      Abrir Visual Studio Code
      Control + shift + p
      Buscar Clone
      Copiar Url Control + V
      Seleccionar Carpeta Donde deseas descargar el repositorio

    Pasos Subir Commits

      Abrir Source Control
      Hacer click en el símbolo + al lado de los archivos que quieres subir
      Click en "Commit"

    Editar Código Desde Github

      Entrar Al repositorio
      Entrar al archivo que deseamos editar
      Realizar cambios necesarios
      Click Commit Change Botón Verde

    Resolución de problemas en Github

      Resolución de problemas en la misma línea de código
      Resolución de problema varios archivos
